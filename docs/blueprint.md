# **App Name**: MultiToolSuite

## Core Features:

- SEO Optimized Tool Listing: Home page featuring a comprehensive, categorized list of free online tools designed for maximum SEO visibility.
- Online Word Counter: Accurately counts words and characters, providing essential metrics for writers and editors. Realtime counter leveraging a tool powered by generative AI will provide immediate feedback of your word's meaning and feeling!
- Image Compressor: Reduces image file sizes while maintaining quality, optimizing images for web use and faster loading times.
- QR Code Generator: Generates QR codes from user-provided text or URLs, supporting various customization options. You can also choose a background and foreground to be used when generating the image
- Text Case Converter: Converts text between different cases (uppercase, lowercase, title case, etc.), simplifying text editing tasks.
- JSON Formatter: Formats and validates JSON code for improved readability and error detection.
- Contact Form Submission: Capture messages via the contact form, and store them in the Firestore database
- Blog System: An SEO-optimized blog to publish articles related to the use of the tool and general advice

## Style Guidelines:

- Primary color: #2E9AFE (a vibrant, professional blue, in the style of modern tech interfaces)
- Background color: #EBF5FB (a light, desaturated blue tint, in keeping with the blue primary)
- Accent color: #2EFE9A (a bright, high contrast green analogous to the blue, intended for CTA buttons, highlights, etc.)
- Body and headline font: 'Inter', a sans-serif font that gives a machined, objective, neutral look
- Simple, line-based icons that communicate functionality clearly without distraction
- Clean, grid-based layout that places a Google AdSense ad to the right of the main content. Ad placement must be compliant with AdSense policies
- Subtle animations, such as fade-ins, that improve UX without slowing down page speed
